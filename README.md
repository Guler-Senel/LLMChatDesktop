*Kural Tabanlı Masaüstü Chatbot
Açıklama

C# ve Windows Forms (.NET Framework) kullanılarak geliştirilmiş, kural tabanlı bir masaüstü chatbot uygulamasıdır. Kullanıcıdan alınan mesajlar anahtar kelimelere göre analiz edilerek önceden tanımlanmış cevaplar üretilir. Yapay zekâ veya harici API kullanılmamıştır.

Kullanılan Teknolojiler

C#

.NET Framework

Windows Forms

Visual Studio

Özellikler

Mesaj gönderme (Buton / Enter)

Anahtar kelime eşleştirme

Kişiselleştirilmiş cevaplar (kullanıcı adı)

Sohbet geçmişi kaydı

Menü desteği (Temizle / Çıkış / Hakkında)

Çalıştırma

Projeyi Visual Studio ile açın

Uygulamayı çalıştırın

Chatbot ile mesajlaşın
